# CONTRIBUTING_AI.md
**Last Updated:** 2026-02-08
**Purpose:** Drop-in contribution rules that make AI changes safe.

## Non-negotiables
- `CODEX_CONSTITUTION.md` is binding
- PR per Intent Unit
- ≤ 5 files touched unless explicitly approved
- No new dependencies unless explicitly approved
- No “done” without proof (tests output / screenshots)

## Required in every PR description
- Outcome + success criteria
- Scope (files touched)
- Tests run (commands)
- Screenshots (if UI/E2E)
- Risks + mitigations
